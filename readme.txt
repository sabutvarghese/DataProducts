Mortgage Calculator
This is a Mortgage Calculator developed using Shiny package in R.

Inputs for this Mortgage calculator are:
  Loan Amount
  Annual Percentage Rate (APR)
  Loan Term in Months
  Property Tax Rate

Outputs
  The outputs of this calculator is the amortization schedule
